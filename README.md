# Infrastructure_as_Code-Orchestration-MediaWiki-via-Docker-Ansible
Ce projet démontre la mise en œuvre d'une infrastructure conteneurisée pour MediaWiki, où l'agilité de Docker rencontre la puissance de configuration d'Ansible. L'approche adoptée est celle du "Tout-Automatisé", du déploiement des conteneurs à la sécurisation des services.
